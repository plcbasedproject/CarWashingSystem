  • Objective of the project

    This project aims to design and implement a car washing system using Programmable logic controller (PLC).

  • Abstract

    A PLC is a digital computer without a mouse or a keyboard, or a monitor. Programmable Logic Controller (PLC) focuses on automating different electro-mechanical processes taking place in various industries. It has specially designed and implemented controllers with a well-programmed microprocessor that are immune to extremely high and low temperatures, humidity, dust, etc.

  • Conclusion

    In this project, we have design and built a car washing system using PLC. We have tested this project. We have noticed different errors in the mechanical frame, programming and the operation of relays. We have changed dimensions of the mechanical frame, Debugged the code and changed the relay .

  • Future Work

    In future ,this system may be depend on (micro controller) which would be compact ,fast and accurate . This system may not have the features and reliability of the original design. It is only being developed to ensure that the design is feasible, not impractical and can be implemented on a much larger scale in a more efficient way.
